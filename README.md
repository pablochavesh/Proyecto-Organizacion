# Gestor de Agenda de Contactos en C++

Este proyecto implementa un sistema para gestionar contactos utilizando diversas estructuras de datos en C++, como parte del curso "Organización archivos y estructuras de datos"

# Descripción
El sistema permite:
- Agregar contactos
- Listarlos ordenadamente
- Buscar por nombre o teléfono (búsqueda secuencial y binaria)
- Eliminar contactos (usando lista enlazada)
- Mostrar historial (usando pila)
- Manejar contactos pendientes (usando cola)
- Mostrar recorrido de contactos en un árbol binario de búsqueda (BST)


# Requisitos técnicos
- Lenguaje: C++
- Sistema operativo: Windows / Linux / Mac
